# Inventory Stock Level Audit

## Project Objective

Analyze inventory data to monitor stock levels, pricing, product categories, and inventory distribution.

## Tools Used

- LibreOffice Calc
- GitHub

## Skills Demonstrated

- Data Cleaning
- Data Analysis
- KPI Creation
- Dashboard Design
- Data Visualization
- Inventory Analysis

## KPIs

- Total Products
- Average MRP
- Maximum MRP
- Average Discount Percentage
- Average Quantity
- Average Selling Price
- Out of Stock Products

## Visualizations

- MRP vs Selling Price
- Quantity vs Selling Price
- Weight vs Quantity
- Category Distribution

## Key Insights

- Product categories are distributed across multiple inventory groups.
- Discounts significantly affect selling prices.
- Inventory quantities vary considerably by product.
- Category analysis helps identify stock concentration.
